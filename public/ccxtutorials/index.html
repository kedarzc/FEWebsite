Finite elements using CalculiX

Description

Calculix is an excellent non-linear solver but lacks in a easy to use pre and post processor. I use Mecway both as pre and post processor. Sometimes I use the natively shipped post processor cgx.

  • Spring between node and ground (Spring1 element)
  • Spring between two nodes acting in a fixed direction (Spring2 element)
